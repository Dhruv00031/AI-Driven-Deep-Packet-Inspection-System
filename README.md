# AI-Driven Deep Packet Inspection System

An AI-powered network security platform capable of capturing, inspecting, analyzing and classifying network packets in real time using Machine Learning and Deep Packet Inspection techniques.

---

## Features

✔ Real-time Packet Capture

✔ Deep Packet Inspection

✔ SQL Injection Detection

✔ XSS Detection

✔ AI-based Traffic Classification

✔ Live Dashboard

✔ Packet Analytics

✔ Attack Logging

✔ MongoDB Storage

✔ REST API
